# Power-BI-projects
Power BI Portfolio: Business Intelligence & Data Visualization
This repository contains my projects created while studying the Power BI business intelligence course. My portfolio includes everything from data extraction, cleansing, profiling, building complex data models to design interactive dashboards for executive decisions.

Tech Stack & Core Competencies
While working on those projects, I used:

Power Query (ETL): Connect to different sources (Excel, CSV, Web), cleanse data, profile data, use M-language for transformation.

Data Modeling: Create star or snowflake schema, define 1-to-many relationship, configure cross-filter directions.

DAX (Data Analysis Expressions): Write calculated columns and measures. For example, CALCULATE, FILTER, and Time Intelligence Functions like YTD, YoY, and SamePeriodLastYear.

Data Visualization: Design Professional UI/UX, create Bookmarks to navigate dashboards, use Drill-through report and create Custom Tooltips.

My Approach
For each project in this repository, I applied the following steps in my analysis process:

Requirement Analysis: Define main business questions and KPIs.

Data Preparation: Manage data types, remove duplicates and handle NULL values in Power Query.

Data Modeling: Define tables, Fact & Dimensions and build centralized Date table (Calendar).

Measures Development: Use DAX expressions to provide flexible analysis of your data.

Visualization: Build visual hierarchy from Summary Tables to Detailed Analysis.
